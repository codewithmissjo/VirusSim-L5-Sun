1. Create a main.py file.

2. Go to (top menu) Terminal and select New Terminal.
    * (Windows only) If the terminal opens in Powershell, you need change it to Command Prompt.
    * To do this in the terminal area, click on the down arrow (v) next to the plus sign and select Command Prompt from the dropdown.

3. Install pygame by typing the following into the Command Prompt terminal:

(for Windows) python -m pip install pygame
(for MacOS) python3 -m pip install pygame
